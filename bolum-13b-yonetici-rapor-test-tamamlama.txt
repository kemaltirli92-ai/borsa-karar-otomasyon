API UCLARI

Web sitesi ve mobil uygulama Endeks Skoru verisini ortak API uzerinden alacaktir.

Gerekli API uclari:

GET /api/index-score/latest

Son yayinlanmis Endeks Skorunu getirir.

GET /api/index-score/history

Tarihsel Endeks Skorlarini getirir.

GET /api/index-score/{run_id}

Belirli bir calismanin tum detaylarini getirir.

GET /api/admin/index-score/{run_id}/metrics

Yonetici icin tum ham metrikleri ve katkilarini getirir.

GET /api/admin/index-score/config

Aktif agirlik ve esik ayarlarini getirir.

PUT /api/admin/index-score/config

Yetkili yoneticinin agirliklari ve esikleri degistirmesini saglar.

POST /api/admin/index-score/recalculate

Belirli bir tarih veya veri surumu icin yeniden hesaplama yapar.

POST /api/admin/index-score/publish

Dogrulanmis Endeks Skorunu yayinlar.
